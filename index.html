Map of American Income and Poverty in 2013

Data Products Course Project

Andrew Reynolds

What the app does

The app allows the user to generate a choropleth map of several economic indicators across the 48 contiguous states, by county. The indicators are

  • Median Household Income
  • Per Capita Income
  • Percent in Poverty (under 18)
  • Percent in Poverty (all ages)
  • Percent in Deep Poverty (all ages)
  • Percent in Deep Poverty (under 18)
  • Total in Poverty (under 18)
  • Total in Poverty (all ages)

Poverty status thresholds vary by family size, number of children, and age of householder. The deep poverty threshold is half of the poverty threshold. See the documentation.

How to use the app

The user creates the map in three steps:

  1. Select an indicator from a dropdown menu.
  2. Select a range of values which is of interest by manipulating a slider.
    • The slider is itself reactive. The available values depend on the indicator selected. Percentage indicators range from 0 to 100. All other indicators range from 0 to the maximum value of the indicator.
    • The color scheme used is also dependent on the indicator selected.
  3. Click the "Go!" button to draw the map. This takes a few seconds.

Once the map is drawn, the user can zoom in by dragging the mouse cursor to select a viewing window. The user must then click "Go!" again to redraw the map. To zoom back out, the user clicks "Go!" with no viewing window selected.

Data and packages

  • The data used to produce the choropleth map comes from the US Department of Agriculture and is available for download here. Only Income.csv was used in this app.
  • Documentation for the data used can be found here.
  • The app uses the ggplot2 and maps packages and was made using the shiny package.
  • This presentation was made using the slidify package.
